## Running Locally

This application requires Node.js v16.13+.

```bash
git clone https://github.com/BogdanBarjaktarevic/space-tourism.git
cd space-tourism
npm install
npm run dev
```

Create a `.env`. Example:

```bash
DATABASE_URL="postgres://..."
SHADOW_DATABASE_URL="postgres://..."
```
